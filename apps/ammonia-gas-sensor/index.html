Ammonia Gas Sensor Monitoring Application

A dedicated app page under the apps route for image-based ammonia sensing and concentration prediction.

Overview

This web application analyzes captured sensor images and estimates ammonia concentration directly in ppm using image-based processing. It is designed for fast and practical usage in sensing workflows.

Application Link: https://ammonia-gas-sensor.streamlit.app/

Workflow

  1. Open the application from the direct link or QR code.
  2. Upload the captured sensor image.
  3. Automatic analysis detects color and intensity response.
  4. The predicted ammonia concentration is shown in ppm.

Key Features

  • Automatic concentration prediction
  • Real-time analysis
  • Direct ppm output display
  • User-friendly interface
